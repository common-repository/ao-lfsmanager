== Description ==

Das Plugin Lite - Font & Style Manager ist die perfekte Lösung für alle, die sich der aktuellen Abmahnwelle, bezüglich Google-Schriftarten entziehen möchten! 
Das Plugin entfernt die Google Fonts automatisch und ersetzt diese durch Schriftarten, die im Theme oder alternativ lokal eingebunden sind. 
So können Sie sicher sein, dass Ihre Seite rechtlich einwandfrei ist. Die Kompatibilität zu Page Buildern und gängigen Themes ist gewährleistet!

Font & Style Manager ist ein schlankes Plugin und nach der Installation sehr einfach zu bedienen. 
Darüber hinaus bietet das Plugin in der Basis-Edition, viele weitere nützliche Funktionen, wie das Zusammenführen von CSS-Dateien, 
die Bereitstellung eines Style Caches, der CSS-Dateien in einen Zwischenspeicher legt, um die Performance Ihrer Seite zu erhöhen. 
Zudem wird durch Font & Style Manager die SEO-Freundlichkeit Ihrer Seite erhöht!

In der Font & Style Manager Agency-Version erhalten Sie zusätzlich die Kontrolle darüber, 
welche Dateien konkret vereint werden sollen und Sie haben auch die Möglichkeit noch manuell Passagen aus Dateien heraus zu löschen.

Laden Sie sich jetzt unser Plugin Font & Style Manager herunter, 
um Ihre Webseite abmahnsicher und zukunftsorientiert zu optimieren.

The plugin automatically removes the Google Fonts and replaces them with fonts that are integrated in the theme or alternatively locally.

== Installation ==

1. Upload `ao-lfsmanager` to the `/wp-content/plugins/` directory or pull it from git as a submodule
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Just activate the checkbox on the settings page and enjoy.

== Frequently Asked Questions ==

= What i need to do, if google fonts are still appearing? =

Google fonts will be loaded when you are logged in, please log out and try again.
If there are still appearing, send us a support mail. Use the button on the settingspage.
